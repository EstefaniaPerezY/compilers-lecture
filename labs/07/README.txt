Compilation Instructions:
1. First, clean any existing compiled files:
    make clean
2. Then, compile the Chatbot:
    make
3. Finally, execute the Chatbot:
    ./chatbot

Dependencies:
The Chatbot is linked with the libcurl library. Make sure you have it installed on your system.
- Installing libcurl on Linux:
    sudo apt-get update
    sudo apt-get install libcurl4-openssl-dev
- Installing libcurl on macOS with Homebrew:
    brew install curl

Usage:
Once the Chatbot is running, you can interact with it by typing commands such as greeting, asking for the time or weather, or requesting movie information.
